# LexiAssist - AI-Powered Legal Practice Management System

LexiAssist is a comprehensive legal practice management system designed specifically for Nigerian lawyers. It combines artificial intelligence with essential practice management tools to streamline legal workflows.

## Features

- AI-powered document drafting and legal analysis
- Legal research with web search integration
- Case management and tracking
- Court calendar with hearing reminders
- Document templates library
- Client information management
- Billing and time tracking system

## Technologies Used

- React 18
- Tailwind CSS
- Claude AI API (Anthropic)
- Browser Storage API

## Setup Instructions

1. Clone this repository
2. Open index.html in a web browser
3. The application runs entirely in the browser

## Usage

The application provides seven main modules accessible through the navigation tabs. Each module is designed to support different aspects of legal practice management in accordance with Nigerian legal standards.
